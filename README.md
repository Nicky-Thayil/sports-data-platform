# Apex

A cloud-native sports data platform serving F1 and Premier League data through a REST API.

## Overview

Apex ingests F1 and Premier League data on a scheduled basis, stores it in PostgreSQL, and serves it through a versioned API with caching, authentication, and request tracing.

## Tools & Technologies

- Python, FastAPI, PostgreSQL, Redis, SQLAlchemy
- Docker, GCP Cloud Run, Terraform, GitHub Actions
- React, Vite
